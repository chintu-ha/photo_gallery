# photo_gallery
Introduction

The Simple Photo Gallery Website is designed using only HTML and CSS.
It allows users to view and explore a collection of images in a clean layout.
The project focuses on learning the basics of web structure and styling.
It demonstrates how to create responsive and organized photo displays.
This website is simple, lightweight, and easy to customize.

<br>
#Technologies Used<br>
HTML5 for structure<br>
CSS3 for styling and layout<br>
<br>


#Functional Requirements
Image Display:-
The system must display images in a grid or column layout.
<br>
Responsive Design:-
The layout must adjust to mobile, tablet, and desktop screens.
<br>
Navigation Menu:-
Must include links like Home, Gallery, About, Contact.
<br>
Hover Effects:-
CSS hover effects such as zoom or shadow must be applied.
<br>
Image Captions (Optional):-
Each image may include a title or short description.
